# ElementaryTicTacToe
My First App.

### What is it?
A tale as old as time,  
It's Tic-Tac-Toe, A Classic!   

### How does the game work?
Take turns setting an 'X' or an 'O'.  
The first to get three in a row wins. 

### Technologies Used
1. Swift
2. UIKit
